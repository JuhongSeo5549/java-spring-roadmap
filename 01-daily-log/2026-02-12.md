## 오늘 한 것
- 강의: Java - 반복문
- 실습: while문, do-while문,break,continue, for문, 중첩 반목문
- 정리:
1. while
조건을 먼저 검사하고, 조건이 true인 동안 반복한다.
조건이 처음부터 false이면 한 번도 실행되지 않는다.
반복 횟수가 명확하지 않거나, 종료 시점이 상황에 따라 달라질 때 유리하다.

2. do-while
코드를 먼저 1번 실행한 뒤 조건을 검사한다.
조건이 false여도 “최소 1번은 실행”된다.
처음 1회 실행이 반드시 필요할 때 사용한다.

3. break / continue
break: 반복문을 즉시 종료하고 밖으로 나간다.
continue: 현재 반복의 나머지 코드를 건너뛰고 다음 반복으로 넘어간다.
(while / do-while / for 모두 사용 가능)

4. for
반복 횟수가 정해져 있을 때 주로 사용한다.
초기화, 조건검사, 증감이 한 줄에 모여 있어 구조가 명확하다.
카운터 변수(반복 횟수 조절 변수)가 눈에 잘 보인다.
for의 3요소(초기식/조건식/증감식)는 선택이며, 모두 생략하면 무한 반복이 될 수 있다.

5. for vs while 선택 기준
for: 정해진 횟수만큼 반복(카운터 기반 반복)
while: 반복 종료 시점이 불명확하거나 조건이 흐름에 따라 바뀌는 반복

6. 중첩 반복문
반복문 안에 또 반복문을 넣을 수 있다.
전체 반복 횟수는 대체로 “바깥 반복 횟수 * 안쪽 반복 횟수”로 증가한다.
