## 오늘 한 것
- 강의: Java - Scanner
- 실습: Scaaner 훈련
- 정리:
1. Scanner로 입력받기
System.in 입력을 편하게 받기 위한 도구가 Scanner이다.
주요 입력 메서드
nextLine() : 한 줄(문자열) 입력(엔터 전까지)
nextInt() : 정수 입력
nextDouble() : 실수 입력
주의: 타입에 맞지 않게 입력하면 InputMismatchException 발생 가능.

2. print vs println
print: 줄바꿈 없이 출력(입력 안내 문구에 자주 사용)
println: 출력 후 줄바꿈(개행 \n 포함)

3. 입력 + 반복 패턴(실무/문제 공통 구조)
무한 반복 while(true)로 계속 입력받고,
특정 값(종료 키워드/0/-1 등) 입력 시 break로 종료한다.

대표 종료 패턴
문자열: "exit" 또는 "종료" 입력하면 종료
숫자: 0 또는 -1 입력하면 종료
두 값 모두 0이면 종료 같은 복합 조건 가능

4. 누적 계산 패턴
sum(합계), count(개수) 같은 변수를 두고
입력값을 누적(sum += input, count++)한다.
평균은 double로 계산해야 소수점이 유지된다.

5. 입력 섞어 쓸 때 주의점(중요)
nextInt() 같은 숫자 입력 후에는 줄바꿈(\n)이 남는다.
바로 nextLine()을 쓰면 빈 문자열이 들어갈 수 있어
중간에 nextLine()으로 “줄바꿈 제거” 처리가 필요할 수 있다.
