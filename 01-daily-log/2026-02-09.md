# 2026-02-09

## 오늘 한 것
- 강의:Java - 변수
- 실습: 변수 값 변경, 변서 선언과 초기화, 변수 타입, 변수 명명규칙
- 정리:
1. 변수란
값을 저장할 수 있는 메모리 공간의 이름이다.
저장된 값은 언제든지 변경할 수 있다.

2. 지역 변수
메서드 안에서 선언되는 변수이다.
반드시 개발자가 직접 초기화해야 한다.

3. 컴파일 에러
자바 문법이 틀렸을 때 발생한다.
실행 전에 오류를 알려주는 좋은 에러이다.

4. 변수 초기화
변수를 선언한 후 처음 값을 저장하는 것이다.

5. 기본 데이터 타입
- 정수형 : byte, short, int, long
- 실수형 : float, double
- 기타 : boolean, char, String

- String : 문자열을 다루는 특별한 타입이며 대문자로 시작한다.

- 리터럴 기본 규칙
정수 리터럴의 기본 타입은 int이다.
long 타입은 숫자 뒤에 L을 붙인다.

- 변수 이름 규칙
숫자로 시작 불가
공백 사용 불가
예약어 사용 불가
영문자, 숫자, _, $ 만 사용 가능

- 변수 이름 관례
소문자로 시작
낙타표기법(Camel Case) 사용

- 자바 명명 규칙 요약
클래스 : 대문자로 시작
변수, 메서드 : 소문자로 시작
상수 : 모두 대문자 + 언더바
패키지 : 모두 소문자
