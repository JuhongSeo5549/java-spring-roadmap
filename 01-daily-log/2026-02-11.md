## 오늘 한 것
- 강의:Java - 조건문
- 실습: if, else, else if, switch, 삼항 연산자
- 정리:
1. if / else
if: 조건이 true일 때만 실행
else: if 조건이 false일 때 실행(대안 처리)
* 핵심: 둘 중 하나만 실행되는 구조(참이면 if, 아니면 else)

2. else if
여러 조건 중 “하나를 선택”해야 할 때 사용한다.
위에서부터 순서대로 검사하며, 처음으로 참인 조건 1개만 실행된다.
else는 필요하면 마지막에 추가(없어도 됨)
* 핵심: 조건이 서로 “연관”되어 있고, 결과가 “단 하나”로 결정될 때

3. if 여러 개(독립 조건)
조건들이 서로 독립이고, 동시에 여러 개가 적용될 수 있으면
else if가 아니라 if를 각각 분리해서 써야 한다.
* 핵심: “중복 적용 가능”이면 if 여러 개 / “하나만 선택”이면 if-else if

4. 중괄호 {} 생략
한 줄만 실행할 때 생략 가능하지만,
가독성/유지보수 때문에 실무에서는 {} 쓰는 것을 권장한다.

5. switch 문
특정 값(동일 비교)에 따라 분기할 때 사용한다.
if보다 간결해질 수 있다.
break가 없으면 아래 case까지 이어서 실행될 수 있다(fall-through).
default는 else 역할(없어도 됨)
* 핵심: “값이 같은지”로만 분기 가능(범위/복합조건은 if가 유리)

6. (참고) Java 14+ switch 표현식
-> 형태로 더 간결하게 작성 가능
값을 반환하는 형태로도 사용 가능

7. 삼항 연산자 (조건 연산자)
단순히 조건에 따라 “값 하나를 선택”할 때 사용한다.
구조: (조건) ? 참일때값 : 거짓일때값
* 핵심: 짧은 값 선택에는 좋지만, 복잡한 로직은 if가 더 읽기 쉽다.
