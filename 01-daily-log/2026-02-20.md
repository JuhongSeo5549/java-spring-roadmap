## 오늘 한 것
- 강의: Java - 생성자,패키지
- 실습: this, 기본생성자, 생성자 - 오버로딩과 this(), import, 패키지규칙
- 정리:
생성자: 객체 생성 시 필드를 초기화하는 특별한 메서드
클래스 이름과 동일, 반환 타입 없음
new 키워드와 함께 호출됨
기본 생성자: 생성자를 정의하지 않으면 자동 제공
생성자 오버로딩 가능
생성자를 통해 “필수 값 강제 초기화” 가능

클래스 이름 충돌 방지
import로 다른 패키지 클래스 사용
패키지명 관례: 모두 소문자, 도메인 역순
구조화된 프로젝트 관리에 필수
