접근 범위를 제한하여 캡슐화 구현
종류
public: 어디서나 접근 가능
protected: 같은 패키지 + 상속 관계
default: 같은 패키지
private: 해당 클래스 내부만

메모리 영역

힙 영역: 객체, 인스턴스
스택 영역: 메서드 호출, 지역 변수

static
클래스 단위로 하나만 생성
