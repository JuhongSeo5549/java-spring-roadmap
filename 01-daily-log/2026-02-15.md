- 정리:
1. 1차원 배열: 선언/생성/초기화
- 선언(아직 배열 없음)
int[] students;

- 생성(크기 지정, 기본값 자동 초기화)
students = new int[5]; // int는 0으로 초기화

- 값 대입/사용(인덱스는 0부터)
students[0] = 90;
System.out.println(students[0]);

- 인덱스 범위 초과 시
ArrayIndexOutOfBoundsException

- 길이
students.length // 배열 요소 개수

- 생성+초기화(긴 버전)
int[] students = new int[]{90, 80, 70, 60, 50};

- 생성+초기화(짧은 버전: 선언과 “같은 줄”에서만 가능)
int[] students = {90, 80, 70, 60, 50};

2. 배열 리팩토링 핵심
- 배열을 쓰면 반복문으로 출력/합계/평균을 “길이 기반”으로 처리 가능
int total = 0;
for (int i = 0; i < students.length; i++) total += students[i];
double avg = (double) total / students.length;

3. 기본형 vs 참조형(배열 포인트)
int, double 같은 기본형: 값 자체 저장
배열(int[] 등) 참조형: “배열이 있는 주소(참조값)”를 저장
배열은 크기가 실행 중에 결정될 수 있어(동적 할당) 참조형 구조를 사용

4. 2차원 배열(행/열)
- 선언/생성
int[][] arr = new int[2][3]; // 2행 3열

- 접근
arr[행][열]

- 중첩 반복으로 출력(길이 기반이 정석)
for (int row = 0; row < arr.length; row++) {
  for (int col = 0; col < arr[row].length; col++) {
    System.out.print(arr[row][col] + " ");
  }
  System.out.println();
}

- 2차원 초기화(가독성 좋게 줄바꿈)
int[][] arr = {
  {1, 2, 3},
  {4, 5, 6}
};

5. 값 “자동 채우기” 패턴
int[][] arr = new int[2][3];
int v = 1;
for (int r = 0; r < arr.length; r++) {
  for (int c = 0; c < arr[r].length; c++) {
    arr[r][c] = v++;
  }
}

6. 향상된 for(for-each)

인덱스 필요 없고 “값만” 순회할 때 간단
for (int n : numbers) {
  System.out.println(n);
}
인덱스(i)가 필요하면 일반 for가 더 적합

합/평균/최대/최소: 초기값을 numbers[0]로 잡고 1부터 순회
